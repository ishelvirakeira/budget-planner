This is a budget management application that helps users track expenses across categories. Authenticated users can add, view, and delete expenses, providing a simple personal finance tracking system.

Find the live demo at https://budget-planner-dy5c.onrender.com

Lessons Learned:

- How to structure a full-stack web app with Node.js, Express, and MongoDB.
- Working with EJS templates to render dynamic content on the frontend.
- Connecting backend data models (MongoDB) with frontend views.
- Understanding basic CRUD operations in a web application context.
